Basic-Operating-System-BOS-
BOS — 一个运行在 TurboWarp 上的简易操作系统内核原型。

🧩 当前状态
文件系统：基于列表的虚拟文件系统（能用，但有 bug 正在修）

内存管理：下标访问式内存

进程管理：正在实现

由一位 12 岁开发者独立编写，目前代码量约 2700 个 Scratch 积木（当然你看到的时候可能已经不是这个数了）。

💻 系统特性
看起来像 16 位系统（内存寻址 16 位）

内存：64 KB（一条 16 位内存条）

CPU：系统自带 CPU（TurboWarp 扮演）

硬盘：约 1 MB（已接近 TurboWarp 列表的极限，不一定能存到这个数）

⚠️ 配置仅供参考，请以实际运行情况为准。

📌 详细简介
BOS 是一个用 Scratch 积木搭出来的操作系统，跑在 TurboWarp 里。
它的诞生是为了搞明白“操作系统到底是怎么工作的” —— 于是就有了这个充满 bug、但确实在跑的玩具内核。

目前你能用它：

读写文件（虽然偶尔会找不到路径）

分配和释放内存（虽然要自己小心别越界）

体验 16 位系统的复古感

如果你也好奇操作系统底层，或者想看看一个 12 岁孩子能写出什么东西，欢迎来逛逛。
当然，代码里还有不少坑，但每个坑都是我亲手挖的，也正在亲手填。
