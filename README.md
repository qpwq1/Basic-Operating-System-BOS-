# Basic-Operating-System-BOS-
BOS - Basic-Operating-System  一个运行在 TurboWarp 上的简易操作系统。目前已实现：  基于列表的虚拟文件系统（有 bug 正在修）  下标访问的内存管理（比文件系统更糟）   进程管理尚未实现，计划后续加入。  由一位 12 岁开发者独立编写，目前代码量约 2700 个 Scratch 积木（当然你看到的时候可能不是这个数了）。

它目前看起来像16位系统（因为内存是十六位的）
配置：
内存：一条64KB内存，指针无法实现1：1（一个字节一个字节的地址存储），只能实现1：2
CPU：系统CPU
硬盘：理论50*20，大约1MB硬盘（看起来不多，实际上已经搭配大TurboWarp的极限了）

配置不一定是最新的，请以实际情况为准
